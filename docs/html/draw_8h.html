Draws an image on the SSD1306 display.

Parameters
dataPointer to the image data (bitmap).
sizeSize of the image data in bytes.
xX-coordinate of the top-left corner of the image.
yY-coordinate of the top-left corner of the image.

This function uses the ssd1306_bmp_show_image_with_offset function from the ssd1306 library to draw a bitmap image on the display at the specified coordinates.
