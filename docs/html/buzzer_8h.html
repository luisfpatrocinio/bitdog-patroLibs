Detailed Description

Buzzer control library for RP2040 (BitDogLab)

Author
Luis Felipe Patrocinio (https://github.com/luisfpatrocinio/)

Provides functions to control a buzzer using PWM on the RP2040 microcontroller. Supports initialization, simple beeps, and tone playback with specified frequency.

Definition buzzer.c:80

Plays a tone with specified frequency and duration.

Generates a PWM signal at the requested frequency with 50% duty cycle. Implements frequency calculation based on RP2040's clock system.

Parameters
frequencyTarget frequency in Hertz (50-5000Hz recommended)
duration_msTone duration in milliseconds
Note
Actual frequency may vary slightly due to integer rounding
Warning
Frequency accuracy decreases at higher frequencies
Blocks execution during tone playback

@implnote Calculation:

top = (CPU_CLOCK / (frequency * divider)) - 1
